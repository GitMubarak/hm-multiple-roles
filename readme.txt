== Installation ==

Install HM Multiple Roles Plugin as a regular WordPress plugin. Here are different ways to install plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search HM Multiple Roles Plugin
3. Click to install
4. Activate the plugin
5. Set the widget in Admin Widgets area.

**Install by upload plugin:**

1. Download the latest version of HM Multiple Roles Plugin (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded hm-multiple-roles.zip and click on "Install Now" button
6. Activate the plugin
7. HM Multiple Roles Plugin will appear in Admin Widget area.
